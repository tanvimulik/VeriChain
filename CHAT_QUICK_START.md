# 💬 Chat System - Quick Start Guide

## 🚀 How to Use the Chat Feature

### For Buyers:

#### Method 1: From Marketplace
1. Go to **Marketplace** (`/marketplace`)
2. Find a crop you're interested in
3. Click the **💬 green chat button** on the crop card
4. Chat window opens automatically
5. Start chatting with the farmer!

#### Method 2: From Dashboard
1. Go to **Buyer Dashboard**
2. Click **"My Chats"** card
3. See all your conversations
4. Click on any chat to continue

### For Farmers:

1. Go to **Farmer Dashboard**
2. Click **"My Chats"** card (or go to `/chats`)
3. See all buyer conversations
4. Click on any chat to reply

---

## 💡 Chat Features

### In the Chat Window:
- **Send Messages**: Type and click 📤
- **Call Farmer**: Click 📞 to call their phone
- **Auto-Refresh**: New messages appear every 3 seconds
- **Scroll**: Automatically scrolls to latest message
- **Crop Context**: See which crop you're discussing

### In the Chat List:
- **Unread Count**: See number of unread messages
- **Last Message**: Preview of last message
- **Time**: When last message was sent
- **Sort**: Most recent chats appear first

---

## 🎯 Typical Conversation Flow

### Example Chat:

**Buyer**: Hello! Is the Nashik Red Onion still available?

**Farmer**: Yes, I have 500kg available. Grade A quality.

**Buyer**: What's your best price for 200kg?

**Farmer**: ₹26/kg for 200kg. I can deliver to Pune FPO.

**Buyer**: Great! I'll send an order request now.

**Farmer**: Perfect! I'll accept it once I receive it.

## ⚠️ Important Rules

### DO:
✅ Discuss crop details, quality, delivery
✅ Negotiate price and quantity
✅ Ask questions about the crop
✅ Agree on delivery location
✅ **Complete payment through FarmConnect**

### DON'T:
❌ Share personal bank details
❌ Make payments outside the platform
❌ Share sensitive information
❌ Spam or harass other users

## 🆘 Troubleshooting

### Chat button not working?
- Make sure you're logged in as a Buyer
- Check your internet connection
- Refresh the page

### Messages not sending?
- Check your internet connection
- Make sure you're logged in
- Try refreshing the page

### Can't see new messages?
- Wait 3 seconds (auto-refresh)
- Or refresh the page manually
